# Hospital-Resource-Dashboard
Hospital Resource Utilization and Patient Outcomes Dashboard

# This project delivers an interactive Power BI dashboard that provides:
- Department-level drill-down
- Predictive alerts for ICU & bed shortages

# Tools
- Power BI (Visualization)
- CSV-based ETL
- On-Premise compatible

# Key KPIs
- Average Length of Stay (ALOS)
- Bed & ICU Occupancy Rate
- Emergency vs Scheduled Admissions
- Doctor Utilization
- Cost per Patient

# Features
- Drill-down by department, age group, branch
- alerts
- Predictive capacity planning
- Exportable reports (CSV / PDF)
